**Sewsafe** is a specialized digital measurement management tool currently in development. It is designed specifically for fashion designers and tailors to securely store, track, and manage client measurements, eliminating the reliance on physical, easily lost notebooks.

## 🚧 Current Development Status
This project is under active development. 
* **Implemented:** Core UI layouts for client profiles, dynamic measurement input logic, and navigation architecture.
* **In Progress:** Integrating the Supabase backend for data persistence and user authentication.

## 🛠 Tech Stack & Architecture
* **Framework:** Flutter
* **Language:** Dart
* **State Management:** Riverpod (Chosen for robust, scalable, and reactive state handling across complex measurement forms).
* **Backend:** Supabase (PostgreSQL)

## ✨ Core Features (Roadmap)
- [x] Adaptive UI for seamless use on both tablets and mobile devices.
- [x] Dynamic input fields tailored for male and female measurement parameters.
- [ ] Secure cloud synchronization of client data.
- [ ] Measurement history and version control.
